# Social Media Image Design Guide

## Required Images & Specifications:

### 1. Open Graph Image (og-image.jpg)
- **Size**: 1200 x 630 pixels (Facebook, LinkedIn, WhatsApp)
- **Format**: JPG or PNG
- **Max File Size**: 8MB

### 2. Twitter Card Image (twitter-image.jpg)  
- **Size**: 1200 x 600 pixels (Twitter)
- **Format**: JPG or PNG
- **Max File Size**: 5MB

## Design Template Recommendations:

### Layout Structure:
```
┌─────────────────────────────────────┐
│  [IQ Logo]              SkillShare    │
│                                     │
│      DATA SCIENCE ROADMAP 2026      │
│                                     │
│  Complete ML, AI & MLOps Guide       │
│                                     │
│  🤖 📊 🧠 ⚙️ 🔄                    │
│  ML  | Data Science | Deep Learning  │
│                                     │
│  🚀 From Basics to Production        │
│                                     │
│  [Gradient Background]              │
└─────────────────────────────────────┘
```

### Design Elements:

#### Background:
- **Gradient**: #667eea → #764ba2 (matching your site)
- **Pattern**: Subtle tech/grid pattern overlay
- **Opacity**: 90% background, 10% pattern

#### Typography:
- **Main Title**: "DATA SCIENCE ROADMAP 2026"
  - Font: Bold, modern sans-serif
  - Size: 48px
  - Color: White
  
- **Subtitle**: "Complete ML, AI & MLOps Guide"
  - Font: Regular, sans-serif  
  - Size: 24px
  - Color: rgba(255,255,255,0.9)

- **Tagline**: "From Basics to Production"
  - Font: Italic, sans-serif
  - Size: 20px
  - Color: rgba(255,255,255,0.8)

#### Icons & Elements:
- **Tech Icons**: 🤖 (ML), 📊 (Data Science), 🧠 (Deep Learning), ⚙️ (DevOps), 🔄 (MLOps)
- **IQ Logo**: Top-right corner, 60x60px
- **Decorative Elements**: Circuit board patterns, data visualization elements

#### Color Palette:
- **Primary**: #667eea (Blue)
- **Secondary**: #764ba2 (Purple)  
- **Accent**: #ffd700 (Gold for highlights)
- **Text**: #ffffff (White)
- **Shadows**: rgba(0,0,0,0.3)

## Tools to Create Images:

### Free Options:
1. **Canva** (canva.com)
   - Template: "Social Media Post" → 1200x630px
   - Use gradient backgrounds and tech elements
   
2. **Figma** (figma.com)
   - Create custom design with exact specifications
   
3. **Adobe Express** (express.adobe.com)
   - Free templates for social media

### Quick Canva Steps:
1. Go to Canva → Create Design → Custom Size → 1200x630px
2. Choose "Gradient" background → Purple to Blue
3. Add text elements with fonts above
4. Upload your IQ logo
5. Add tech icons from Canva library
6. Download as JPG

## Content for Both Images:

### Open Graph (1200x630):
- **Focus**: Comprehensive overview
- **Elements**: All tech stack icons
- **Text**: Full title + subtitle + tagline

### Twitter (1200x600):
- **Focus**: Mobile-optimized
- **Elements**: 3 main icons only
- **Text**: Shorter, punchier text

## Testing Your Images:

After creating and uploading:
1. Test with Facebook Debugger: developers.facebook.com/tools/debug
2. Test with Twitter Card Validator: cards-dev.twitter.com/validator
3. Test on WhatsApp by sharing link
